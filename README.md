# Umu - Functional Scripting Language

Umu is:
	Functional programming language -- like LISP, Erlang, ML, Haskell
	Dynamic typing -- like LISP, Erlang, Perl, Python, Ruby


## Installation

    $ gem install umu

## Usage

	$ umu -i
	0001> 1+2

	-> 3 : Int
	0002>
